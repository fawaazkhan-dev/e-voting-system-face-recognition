# E-Voting System with Face Recognition

A secure mobile voting application designed to ensure safe and reliable voting using modern authentication techniques.

## ⚙️ Requirements
- Python 3.x (Anaconda) / Flutter
- MySQL
- IDE (VS Code, Android Studio, etc.)

## ▶️ How to Run

### Backend (Flask)
1. Navigate to backend folder:
   cd backend

2. Install dependencies:
   pip install -r requirements.txt

3. Run the Flask server:
   python app.py



### Frontend (Flutter)
1. Navigate to frontend folder:
   cd frontend

2. Install dependencies:
   flutter pub get

3. Run the app:
   flutter run


### Notes
- Ensure Python and Flutter are installed
- Database (SQL) must be configured correctly

## 🚀 Features
- Face recognition login system
- OTP (One-Time Password) verification
- Secure voting process
- User authentication and data protection

## 🛠 Technologies Used
- Flutter (Mobile App)
- Flask (Backend)
- SQL (Database)
- Python

## 📌 Description
This project was developed as part of my final year dissertation. It focuses on building a secure and user-friendly voting system.

The system combines biometric authentication (face recognition) with OTP verification to improve security and reduce fraud.

## 🔄 How It Works
1. User logs in using face recognition
2. OTP verification is triggered
3. User casts vote securely
4. Vote is stored in SQL database
