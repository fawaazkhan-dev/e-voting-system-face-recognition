# E-Voting System with Face Recognition

A secure mobile voting application designed to ensure safe and reliable voting using modern authentication techniques.

## 🚀 Features
- Face recognition login system
- OTP (One-Time Password) verification
- Secure voting process
- User authentication and data protection

## 🛠 Technologies Used
- Flutter (Mobile App)
- Flask (Backend)
- SQL (Database)
- Python

## 📌 Description
This project was developed as part of my final year dissertation. It focuses on building a secure and user-friendly voting system.

The system combines biometric authentication (face recognition) with OTP verification to improve security and reduce fraud.
